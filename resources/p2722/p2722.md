# [문제 설명](https://codeup.kr/problem.php?id=2722)
5각형을 다음과 같이 그려나간다.<br>
다음과 같은 식으로 5각형이 그려질 때 n번째 5각형의 점의 수를 구하는 프로그램을 작성하시오.

# 해설
n번째 오각형의 점의 갯수를 f(n) 이라 한다면,

| n   | f(n)                          |
|-----|-------------------------------|
| 1   | 5*1 - 0                       |
| 2   | f(1) + 5*2 - 3                |
| 3   | f(2) + 5*3 - 5                |
| 4   | f(3) + 5*4 - 7                |

f(n)의 점화식은 `f(n-1) + (5*n - (2*(n-1)+1))` 이다.
